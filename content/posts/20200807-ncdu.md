[ncdu](https://dev.yorhel.nl/ncdu) (NCurses Disk Usage) is a curses-based version of the well-known `du`, and provides a fast way to see what directories are using your disk space.

<!--more-->

## Install on Fedora

```bash
dnf install ncdu
```
